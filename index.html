Version 2.0 · Année scolaire 2025-2026
Application complète de gestion scolaire pour les établissements du primaire. Gérez élèves, matières, notes, bulletins et classements depuis une interface simple et intuitive. Toutes les données sont stockées localement sur votre appareil — aucune connexion internet requise pour l'utilisation courante.
