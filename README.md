## Version Desktop (.exe)

Pour créer une version exécutable Windows :

### Prérequis
- [Node.js](https://nodejs.org/) (v18 ou supérieur)

### Installation

```bash
npm install
```

### Lancer en mode développement

```bash
npm start
```

### Compiler pour Windows

```bash
npm run build:win
```

Les fichiers installables (.exe) seront générés dans le dossier `dist/`.

### Fichiers générés
- `GestionScolairePro-x.x.x-Setup.exe` — Installateur
- `GestionScolairePro-x.x.x-Portable.exe` — Version portable (sans installation)

## Fonctionnalités

- Gestion des élèves et classes
- Saisie et calcul des notes
- Statistiques et bulletins
- Impression des documents
- Mode sombre / clair
- Sauvegarde locale (IndexedDB)
- Export / Import des données
